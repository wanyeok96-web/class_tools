학급 운영 도구 모음입니다. 브라우저에서 `index.html`을 열면 사용할 수 있습니다.

## 문서변환 (한글 → PDF)

허브의 **문서변환**에서 `.hwp` / `.hwpx`를 PDF로 바꿉니다.

브라우저에서 바로 변환됩니다. 파일은 외부 서버로 올라가지 않습니다.
내장 한글 폰트(Noto KR Regular/Bold)와, 가능하면 PC에 있는 함초롬·맑은고딕 등을 함께 사용합니다.

(선택) 한컴 한글이 설치된 PC에서 `converter/실행.bat`을 실행하면, 연결된 동안 해당 엔진으로 더 정밀하게 변환할 수 있습니다.
