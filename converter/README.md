# 로컬 한글 COM 변환기 (권장)

원본 한글 파일과 **가장 비슷하게** PDF로 만들려면 이 변환기를 사용하세요.
한컴 한글 엔진이 직접 저장하므로 글자 위치·모양·서식 충실도가 높습니다.

## 사용

1. 한글(한컴오피스) + Python 3 설치
2. `실행.bat` 더블클릭 (창을 닫지 마세요)
3. Class Tools → **문서변환**에서 「한글 엔진 연결됨」 확인
4. 파일 업로드

주소: `http://127.0.0.1:19531`

변환기가 꺼져 있으면 브라우저 WASM으로 폴백합니다. (서식이 달라질 수 있음)
